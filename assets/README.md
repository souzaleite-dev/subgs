# assets/

Recursos visuais do projeto: diagramas e figuras usadas no `README.md` e no PDF de entrega.

| Arquivo | Origem | Descrição |
|---------|--------|-----------|
| `arquitetura.png` | `python scripts/gerar_diagramas.py` | Diagrama da arquitetura da solução |
| `treino_historico.png` | `python -m src.ml.train` | Curvas de acurácia e perda |
| `matriz_confusao.png` | `python -m src.ml.train` | Matriz de confusão (validação) |
| `amostras_predicao.png` | `python -m src.ml.train` | Exemplos classificados pela CNN |
| `dashboard.png` | *captura manual* | Print do dashboard Streamlit (para o PDF) |
| `logo-fiap.png` | *opcional* | Logo usada na capa do PDF |

> As figuras geradas automaticamente são criadas ao rodar o treino e os scripts. Para o `dashboard.png`, abra o dashboard e tire um print.
